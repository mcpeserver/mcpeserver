# 🎓 RAN DEV Showcase | Arsitek Web Minecraft Indonesia

Koleksi infrastruktur web berperforma tinggi untuk ekosistem komunitas Minecraft Indonesia. Proyek ini dirancang untuk menunjukkan standar baru dalam pembuatan landing page server Minecraft yang cepat, aman, dan estetik.

## 🚀 Fitur Utama

- **Next.js 15 & React 18**: Framework terbaru untuk performa maksimal dan optimasi Edge Runtime.
- **Sistem Monitoring Arsitektur (SMART)**: Dashboard real-time untuk monitor TPS, RAM, dan status infrastruktur.
- **Interactive Developer Console**: Pengalaman CLI langsung di browser untuk kontrol sistem simulasi.
- **Optimasi SEO Mendalam**: Struktur metadata, OG Tags, dan skema JSON-LD untuk peringkat pencarian teratas.
- **PWA Ready**: Siap diinstal di perangkat mobile untuk akses instan tanpa browser.
- **Security Focused**: Dilengkapi dengan header keamanan ketat dan kebijakan perlindungan data.

## 🛠️ Stack Teknologi

- **Frontend**: Next.js 15 (App Router), TypeScript, Tailwind CSS.
- **UI Components**: Radix UI, Lucide Icons, Minecraft-inspired Pixel Framework.
- **Monitoring**: Custom Hooks dengan simulasi metrik real-time.
- **Deployment**: Vercel Edge Network.

## 📦 Instalasi Lokal

1. **Klon Repositori**
   ```bash
   git clone https://github.com/mcpeserver/mcpe-showcase.git
   ```

2. **Instal Dependensi**
   ```bash
   npm install
   ```

3. **Jalankan Server Pengembangan**
   ```bash
   npm run dev
   ```
