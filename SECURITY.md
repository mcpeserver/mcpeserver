# Kebijakan Keamanan (Security Policy)

## Keamanan RAN DEV

Kami mengutamakan keamanan seluruh infrastruktur web yang kami bangun. Jika Anda menemukan kerentanan keamanan pada repositori ini atau proyek yang dikembangkan oleh RAN DEV, harap ikuti panduan di bawah ini.

## Versi yang Didukung

| Versi | Didukung |
| ------- | --------- |
| 2.x     | ✅ Ya      |
| 1.x     | ❌ Tidak   |

## Melaporkan Kerentanan

Harap jangan melaporkan kerentanan keamanan melalui Issue publik di GitHub. Sebagai gantinya, kirimkan laporan Anda melalui:

- **Email**: security@randev.net (Contoh)
- **WhatsApp**: Melalui link kontak di [Website Utama](https://mcpeserver.vercel.app)

Kami akan berusaha memberikan respon awal dalam waktu 48 jam dan memberikan update berkelanjutan hingga masalah teratasi.

## Standar Perlindungan

Proyek ini menerapkan:
- **Content Security Policy (CSP)**
- **XSS Protection**
- **HSTS Enforcement**
- **Dependency Audit Berkala**
